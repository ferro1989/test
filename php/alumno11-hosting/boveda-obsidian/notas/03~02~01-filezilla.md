# Filezilla

**FileZilla** es un **cliente FTP (File Transfer Protocol)** de código abierto que permite **transferir archivos entre un ordenador local y un servidor remoto**. Se utiliza principalmente para **gestionar los archivos de sitios web alojados en servidores**, permitiendo subir, descargar, editar o eliminar contenidos de forma segura.

#### Características principales

- **Compatibilidad con varios protocolos:** FTP, FTPS (FTP seguro mediante SSL/TLS) y SFTP (basado en SSH).
    
- **Gestión de sitios:** permite guardar múltiples configuraciones de conexión con usuario, contraseña, puerto y ruta del servidor.
    
- **Interfaz visual de doble panel:** el panel izquierdo muestra los archivos locales y el derecho los del servidor, facilitando la sincronización mediante arrastrar y soltar.
    
- **Soporte para transferencias múltiples:** puede subir o descargar varios archivos simultáneamente.
    
- **Reanudación de transferencias interrumpidas:** útil en conexiones inestables o archivos grandes.
    

#### Usos comunes

- Subir sitios web creados en **WordPress, Joomla o HTML** a un hosting.
    
- Descargar copias de seguridad desde el servidor.
    
- Editar archivos de configuración remotos (como `.htaccess` o `wp-config.php`).
